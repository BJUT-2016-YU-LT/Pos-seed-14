需求02：
超市为了促销即将过季的商品，需要对这些商品进行打折销售。打折商品的折扣信息如下所示：
{
        barcode: 'ITEM000004',
        name: '电池',
        unit: '个',
        price: 2.00，
        discount: 0.8
 }

在生成最终的购物清单时候，对打折的商品需要按照折扣价计算。
示例：
输入：
[
    {
        barcode: 'ITEM000004',
        name: '电池',
        unit: '个',
        price: 2.00，
        discount: 0.8
    }
]    

输出：
***商店购物清单***
名称：电池，数量：1个，单价：2.00(元)，小计：1.6(元)
----------------------
总计：1.60(元)
节省：0.40(元)
**********************
